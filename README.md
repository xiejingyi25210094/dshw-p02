## 数据来源

## 指数选择
- **沪深 300（000300）**：作为市场基准用于后续的 CAPM 分析，覆盖大盘股、流动性好，常作为国内市场风险溢价的代表。
- **中证 500（000905）**：作为补充指数用于对比中盘股票表现（相比沪深300 更侧重中等市值公司），便于观察不同市值区间的风险与收益差异。

（在 `01_download.ipynb` 中已实现：优先尝试通过 akshare 的成分接口获取成分表，若无则回退到指数历史/日线接口并保存为 `data/index/index_000300.csv` 与 `data/index/index_000905.csv`。）

## 宏观指标选择
- **必选：CPI 同比增速** — CPI 反映通胀水平，通胀预期与利率调控会影响股票估值（尤其是消费与房地产类板块）。
- **自选：M2同比增速** — M2同比增速反映市场流动性的宽松程度。通常 M2 增速上行意味着资金面偏松，部分资金可能流入股市，对权益资产估值形成支撑；反之则可能带来流动性收紧压力。因此该指标有助于从宏观流动性角度解释股票市场的波动。

（在 `01_download.ipynb` 中已实现 CPI 的下载；建议再下载并保存汇率或 M2 数据以补充分析，可在 `02_clean.ipynb` 中与行情数据合并用于回归与时序分析。）

## 如何运行
1. 安装依赖：pip install -r requirements.txt
2. 运行 `01_download.ipynb` 下载原始数据
3. 运行 `02_clean.ipynb` 清洗并保存为 CSV/Parquet
4. 运行 `03_analysis.ipynb` 生成分析图表和 `report.html`
